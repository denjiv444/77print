# 77print | Infographie & Identité Visuelle

77print est un studio de création et d'impression spécialisé dans l'identité visuelle, l'édition et la production print. Ce site web corporate, au design moderne et épuré, met en avant l'exigence et le savoir-faire de l'atelier à travers un portfolio premium.

## 🚀 Fonctionnalités Clés

- **Portfolio Masonry Premium** : Un affichage dynamique des projets avec un rythme éditorial.
- **Glassmorphism UI** : Des overlays transparents et élégants pour la consultation des projets.
- **Redesign Complet** : Une interface moderne centrée sur les visuels haute qualité du studio.
- **Performance Optimisée** : Développement avec Vite et React pour une rapidité d'exécution maximale.
- **Entièrement Responsive** : Une expérience fluide sur mobile, tablette et desktop.

## 🛠️ Stack Technique

- **Frontend** : [React](https://reactjs.org/) + [TypeScript](https://www.typescriptlang.org/)
- **Build Tool** : [Vite](https://vitejs.dev/)
- **Styling** : [Tailwind CSS](https://tailwindcss.com/)
- **Animations** : [Framer Motion](https://www.framer.com/motion/)
- **Icones** : [Lucide React](https://lucide.dev/)
- **Typographie** : Inter & Playfair Display

## ⚙️ Installation Locale

Pour faire tourner le projet sur votre machine :

1.  **Cloner le dépôt** :
    ```bash
    git clone https://github.com/denjiv444/77print.git
    cd 77print
    ```

2.  **Installer les dépendances** :
    ```bash
    npm install
    ```

3.  **Lancer le serveur de développement** :
    ```bash
    npm run dev
    ```

## 🌐 Déploiement

Le projet est configuré pour un déploiement continu sur **Vercel**. Chaque push sur la branche `main` déclenche une nouvelle build et mise à jour en ligne automatiquement.
